## Installation



Requires B9 Aerospace Pack:  
 http://forum.kerbalspaceprogram.com/threads/92630

Requires Firespitter, KM_Gimbal, and RasterPropMonitor, included in B9

Place the Kerbojet folder in your KSP GameData folder.
